Abstract

This is a simple HTML page for the IT 3203 project. The introductory paragraph for my research will go here.

In 1990, Tim Berners-Lee developed the WorldWideWeb and it served as the first interactive “browser”. It allowed users to interact with the web servers efficiently while searching for information on the Internet. Web browsers play a pivotal role in the way people interact online, whether it is researching new topics or communicating with others. Regardless of how they are used, browsers must be simple, functional, and private in order for users to effectively use them.
